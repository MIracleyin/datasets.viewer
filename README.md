## Install

- Requirement.txt

```
streamlit~=1.30.0
opencv-python~=4.9.0.80
numpy~=1.26.3
pillow~=10.2.0
pandas~=2.1.4
matplotlib~=3.8.2
datasets~=2.16.1
wordcloud~=1.9.3
transformers~=4.37.0
```

```
pip3 install -r requirements.txt
```
